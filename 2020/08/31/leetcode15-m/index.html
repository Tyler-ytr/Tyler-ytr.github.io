15. 三数之和

难度中等2455

给你一个包含 n 个整数的数组 nums，判断 nums 中是否存在三个元素 a，b，c ，使得 a + b + c = 0 ？请你找出所有满足条件且不重复的三元组。

注意：答案中不可以包含重复的三元组。

示例：

通过次数293,697

提交次数1,016,765

还是看了题解,排序太香了！

主要操作是排序使用双指针进行检查;同时记得去重;
